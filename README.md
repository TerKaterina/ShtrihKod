# Штрихкод - это...
графическая информация, нанесенная на поверхность, маркировку или упаковку изделия, которая представляет собой последовательность черных и белых полос (или других геометрических фигур) разной ширины.
Основные типы:
- Линейные (например, EAN-13, UPC, Code 128). Это привычные нам полоски. Чем больше ширина полос, тем больше информации они могут хранить, но их объем ограничен.
- Двумерные (QR-код, Data Matrix). Это уже не полоски, а квадраты из точек. Они могут хранить в сотни раз больше информации: ссылки на сайты, визитки, целые абзацы текста.

Для нашего проекта мы будем использовать линейные штрих коды.

# Выбранные библиотеки
Для работы с генерацией и сканированием штрихкодов мы выбрали Zxing и OpenCV соответственно. Далее мы рассмотрим каждую из эих библиотек более подробно.

# OpenCV (cv) - визуализация и сохранение изображения
OpenCV используется для работы с камерой и отрисовки интерфейса: через VideoCapture получаем изображение с камеры, через imshow выводим окно, через rectangle, line и putText рисуем рамку, кнопки и текст.
ZXing-C++ используется для распознавания штрихкода: кадр сначала переводится в оттенки серого с помощью cvtColor, затем создаётся ZXing::ImageView, после чего вызывается ZXing::ReadBarcode(...) для поиска кода на изображении.
Если код найден, через result.text() получаем его значение, а через result.position() — координаты его области на кадре. Дальше программа проверяет, находится ли код целиком внутри основной рамки. Только если это условие выполняется, вокруг него рисуется зелёная рамка и на экран выводится текст распознанного кода.
Дополнительно через OpenCV реализованы кнопки Back и Input и затемнение зоны вне рабочей рамки.
